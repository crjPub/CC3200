# 基于CC3200芯片的wifi控制研究

> 使用TI公司的CC3200芯片，完成基于wifi访问的近场电机控制系统的实现，设计包含网页设计与基于芯片的编程。

## 预期目标
- 基于wifi的网页访问
- 对从机的控制  
- 接受其它主机的控制
